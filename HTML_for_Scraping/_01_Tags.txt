How a Web Scraper Works :-

Website URL
      │
      ▼
requests.get(URL)
      │
      ▼
HTML Source Code
      │
      ▼
BeautifulSoup(html)
      │
      ▼
Find Required Tags
(find, find_all, select)
      │
      ▼
Extract Text/Attributes
(text, href, src)
      │
      ▼
Store in CSV / JSON / Database



Most Common HTML Attributes

+------------+-----------------------------+
| Attribute  | Purpose                     |
+------------+-----------------------------+
| id         | Unique element             |
| class      | Group of similar elements  |
| href       | Link URL                   |
| src        | Image/Video URL            |
| alt        | Image description          |
| name       | Form field name            |
| value      | Input value                |
| type       | Input type                 |
+------------+-----------------------------+
div = no matter how much content it possess, it will take space of a whole page, will occupy whole width of the screen, which means this is a
'block element'
'anchor tag' will only occupy space of its own length, which means it is an 'inline element'
